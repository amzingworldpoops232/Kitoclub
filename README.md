# KITO CLUB

Indie volcanic pixel platformer with PvE rooms and multiplayer lobby.

---

## Play locally

```bash
npm install
npm start
```

Open **http://localhost:3000**

---

## Controls

- **PC:** A/D move, W/Space jump
- **Mobile:** D-pad
- **Chat:** **CHAT** button in-game

## Multiplayer

1. **PLAY** → **PvE**
2. Create or join a room
3. Host presses **START**
